Periop Drug Grid v2

Open index.html locally or deploy the folder as a static website with Cloudflare Pages, Netlify, Vercel, GitHub Pages, or any static host.

Files:
index.html - app shell
styles.css - layout and spreadsheet UI
app.js - calculator, filters, export, drawer
data.js - starter medication database

This is a prototype for education/reference only. The source fields are chapter-level placeholders and dose data must be audited against Miller, package inserts, and local policy before clinical use.
